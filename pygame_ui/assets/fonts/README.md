# UI フォント

## 推奨

[Noto Sans JP](https://fonts.google.com/noto/specimen/Noto+Sans+JP)（SIL Open Font License 1.1）

リポジトリルートで:

```bash
python scripts/download_ui_font.py
```

配置後のファイル名:

- `NotoSansCJKjp-Regular.otf`
- `NotoSansCJKjp-Bold.otf`（任意）
